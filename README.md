# tea-store-experience

An online tea store is made on React.js with a personal customer account, shopping cart and "Favorite Products" menu. The light and dark theme of the application should be implemented using Tailwind CSS

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/tea-store-experience.git
cd tea-store-experience
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
